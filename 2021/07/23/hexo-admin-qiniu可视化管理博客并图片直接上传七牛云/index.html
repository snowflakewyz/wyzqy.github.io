hexo-admin-qiniu可视化管理博客并图片直接上传七牛云

发表于 2021-07-23 更新于 2021-09-07 分类于 技术

前言

在写博客时是通过md文件来编写，不是很方便。hexo-admin-qiniu插件是根据hexo-admin进行改写的。

这两个插件都有一个共同点，都能可视化的管理博客，但在我们写博客时在所难免需要把图片粘贴到博客上，hexo-admin是直接把粘贴在博客上的图片放到了根目录的/source/images文件夹下，上传时也会一起打包上传，缺点显而易见，内容会越来越繁琐。

重点来了，hexo-admin-qiniu插件可以直接把粘贴到博客上的图片上传七牛云，并直接生成图片外链，把图片放在云端，大大减小了文件体积

二: 七牛云使用方法及注意事项

首先注册七牛云账号，在管理控制台中找到对象储存中的空间管理，点新建
空间

新建完仓库，需要绑定自己的域名，否则七牛云会默认绑定一个临时域名，有效期只有30天。绑定完域名，就可以上传自己的图片了

有10个g的空间供你免费使用，放博客的图片是绰绰有余了，把图片存在云端，使博客项目变的不那么庞大，七牛云上不止能存图片，更多功能等着你的探索！
